# Martiliu Packo Product Management System  
**Software Engineering Lab Activity 7**

A lightweight, responsive **Product Management System** built with **HTML, CSS, and vanilla JavaScript** for a hardware store (Martiliu Packo Hardware). Features include user authentication, dashboard analytics, billing with cart functionality, and account management with role-based access.

---

## Features

| Module | Functionality |
|-------|---------------|
| **Login System** | Simple username/password authentication (demo mode) |
| **Dashboard** | Overview with stats: Total Products, Low Stock, Sales, Revenue Growth |
| **Billing & POS** | Search products, add to cart, calculate subtotal/tax/total, process payment |
| **Account Management** | Add new users, edit profiles, view user table with status |
| **Multi-step Modals** | Wizard-style registration and profile update with validation |
| **Toast Notifications** | Success, error, and info messages with auto-dismiss |
| **Responsive Design** | Clean UI with modern cards, icons (Font Awesome), and mobile-friendly layout |
